Commands

  • IAM

    identifies the current user all further commands are executed as the current user until the login is changed

    example: IAM cwheeler

  • OPEN

    opens a connection to another user all further commands are executed against the opened user

    example: OPEN rwestphal
  • mkdir

  • MKEY

    used to create a key

  • hideman hides the manual
  • ipconfig /renew ipconfig /release
  • iamnot logs out specified user IAMNOT cwheeler
